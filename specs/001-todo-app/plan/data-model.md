# 데이터 모델: 오늘의 할 일 웹 앱

## 엔터티

### Todo
- id: 정수형 식별자
- title: 할 일 제목, 최대 100자
- completed: 완료 여부, 기본값 false
- created_at: 생성 시각
- updated_at: 마지막 수정 시각

## 제약 조건

- title은 빈 문자열 또는 공백만으로 구성될 수 없다.
- title은 100자를 초과할 수 없다.
- completed는 boolean 값이어야 한다.
- 삭제 시 해당 Todo가 존재하지 않으면 404 응답을 반환한다.

## 관계

- 단일 사용자용 앱이므로 Todo는 사용자와 직접 연결되지 않는다.
- 모든 Todo는 동일한 저장소에서 관리된다.
